# InfiniteMarquee

A small zero-dependency infinite marquee demo (HTML + vanilla JS).

Features
- Horizontal and vertical marquees
- Pause on hover, reverse on hover
- Drag-to-scroll with inertia
- Edge fade mask
- Auto-init via `data-marquee-*` attributes

Files
- `index.html` — demo page showing several configurations.
- `infinite-marquee.js` — library (UMD) with auto-init.
